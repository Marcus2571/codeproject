HTML "Children"


Hypertext Markup Language (HTML) Består af rigtig mange forskellige tags som fx. div tagget, men i denne lektion vil jeg kun tale om de grundlæggende. som fx. paragraf tagget <p></p>. Paragraf tagget bruges til at lægge tekst ind på en hjemmeside.

html er det programmeringssprog alle hjemmesider på internettet er bygget på. html kode fortæller web browsere hvordan de skal vise indhols som billeder, links, og tekst på en hjemmeside. Html består af start tags < > og slut tags </ >. Når du sætter en html fil op, starter du med, at fortælle browseren, at filen er en html fil med <!DOCTYPE html> tagget. I modsætning til næsten alle andre tags, så skal det her tag ikke bruge et slut tag. Det næste du gør, er du bruger <html> tagget til at fortælle browseren hvor html koden befinder sig. Inde i html tagget bruger du <head> og <body> tags. Head tagget indeholder din hjemmesides titel, det indeholder også links til externe filer som CSS filer hvilket jeg vil komme ind på senere i den her guide. Body tagget indeholder alt det synlige indhold som det her tekst.

Billedet ovenover viser en meget basic html fil. Billedet viser ikke de mere avanceret ting, du bruger når du lære html at kende lidt bedre.
