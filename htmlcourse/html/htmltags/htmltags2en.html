HTML "Children"


Hypertext Markup Language (HTML) consists of a ton of different tags like the div tag, but in this lesson i will only talk about the basic ones. Like the paragraph tag <p></p>. The paragraph tag is used to display text on a website.

html is the language that structures all websites on the internet. html code tells web browsers how to display content like images, links, and text on a website.
html are build of start tags < > and closing tags </ >. When you first setup a html file, you start by telling the browser that the file contains html with the <!DOCTYPE html> tag. Unlike almost every other tag this tag does not need a closing tag. You then use the <html> tag to tell the browser that this section contains html code. Inside the html tag you use the <head> and <body> tags. The head tag contains the title of your website, it also contains the links to external files like CSS files which i will get onto later into this guide. The body tag contains all the visible content like this text.
